# Metrônomo Acessível

Um metrônomo web focado em acessibilidade, navegação por leitores de tela e usabilidade. Desenvolvido como um Progressive Web App (PWA) para funcionar offline e ser instalável em dispositivos móveis.

## 🚀 Funcionalidades
* **Controle de Tempo e Compasso:** Ajuste preciso de BPM (20 a 300) e marcações de compasso (1 a 7 tempos).
* **Web Audio API:** Motor de áudio nativo do navegador para garantir que os bipes saiam cravados no tempo, sem atrasos.
* **Repertório Local:** Sistema de salvamento de músicas usando `localStorage`. Salve, carregue e exclua seus andamentos favoritos sem perder nada ao fechar o app.
* **Acessibilidade em Primeiro Lugar:** Interface construída com HTML semântico, garantindo fluidez e clareza para navegação com NVDA, TalkBack e outros leitores de tela.
* **Suporte Offline (PWA):** Instale no celular e use durante os ensaios mesmo sem internet.

## 🛠️ Tecnologias Utilizadas
* HTML5 Semântico
* JavaScript (ES6+)
* Web Audio API
* Service Workers e Manifest (PWA)
